# Texで数式の=(イコール)のレンダリング位置を揃えたい

```
\usepackage{mathtools}
...
\begin{align*}
emo &= \argmax_{hoge}P(hoge|foo)\\
    &= \argmax_{hoge}\frac{P(foo|hoge)P(hoge)}{P(foo)}\\
    &= \argmax_{hoge}P(foo|hoge)P(foo)
\end{align*}
```
